# @gentleduck/iam-docs

Next.js documentation site for duck-iam, the type-safe RBAC + ABAC access control engine for TypeScript.

## Quick Start

```bash
bun run --filter @gentleduck/iam-docs dev
```

## Scripts

- `bun run --filter @gentleduck/iam-docs dev` -- run the dev server
- `bun run --filter @gentleduck/iam-docs build` -- production build
- `bun run --filter @gentleduck/iam-docs start` -- serve the build
