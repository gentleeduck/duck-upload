## Overview

gentleduck/upload ships agent skills that teach AI coding assistants how to work with the upload engine. Skills follow the open [Agent Skills specification](https://agentskills.io/specification) and work with Claude Code, GitHub Copilot, Cursor, Cline, OpenCode, and 30+ other agents.

## Install

```bash
npx skills add gentleeduck/duck-upload
```

This installs all available skills into your agent's configuration.

## Available Skills

| Skill | Package | What it covers |
|---|---|---|
| `duck-upload` | `@gentleduck/upload` | Upload engine, strategies, store, React hooks, progress tracking, persistence |

## MCP Documentation Server

The docs site exposes an MCP server for real-time documentation access:

```
https://upload.gentleduck.org/api/mcp
```

To connect your agent, add this to your project's `.mcp.json`:

```json
{
  "mcpServers": {
    "duck-upload-docs": {
      "type": "url",
      "url": "https://upload.gentleduck.org/api/mcp"
    }
  }
}
```

## How Skills Work

Each skill is a `SKILL.md` file with YAML frontmatter and markdown instructions. The agent reads the `description` to decide when to activate the skill. The markdown body provides the instructions. Reference files in a `references/` subdirectory provide deeper context loaded on demand.
